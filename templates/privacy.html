User Data

User data, like usernames, passwords, preferences, and users followed are kept locally on the server.

Passwords are hashed on user creation with passlib, which can be found here.

No user data will be ever distributed to third parties, and will only be accessed by administrators if manual correction is needed.

Cookie Policy

This site stores cookies on your local machine for the express purpose of user authentication. Feel free to inspect them.

The module used to do this is called Flask-Login and can be found here.

The data in these cookies is never used for anything else nor ever distributed to third parties.


General Logging

Like any other web service, this one generates logs with the IP addresses of users accessing it. These logs may be periodically inspected for debugging purposes.

As with cookies, no content from these logs will ever be distributed publicly nor to third parties.
